# 🦖 BotMate | Premium AI Agency Platform

> **"Grow your brand digitally with BotMate."**

BotMate is a futuristic, high-performance web platform designed for a next-generation AI agency. Built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**, it features a cinematic interface, deep interactivity, and a fully responsive design system.

![BotMate Banner](/public/botmate-icon.png)

## 🚀 Key Features

### 🎨 **Cinematic UI/UX**
- **Glassmorphism**: Premium frosted glass effects on cards, modals, and navigation.
- **Deep Space Atmosphere**: Custom `GlobalBackground` with dynamic particles, grid lines, and noise textures.
- **Micro-Interactions**: Hover lifts, glowing borders, and magnetic button effects.
- **Cinematic Headers**: Unified `CinematicHeader` component with animated gradient text and ambient orb effects.

### 🤖 **Interactive Systems**
- **Service Showcase**: Zigzag cinematic layout with scroll-triggered reveals and `FuturisticPlaceholder` fallbacks.
- **System Architecture**: Interactive "Core Node" layout illustrating AI agent integration.
- **Local Business Plans**: Holographic 3D tilt cards for pricing plans (Starter, Business, Enterprise).
- **Mission Control**: High-impact "Launch Sequence" CTA.
- **System Evolution**: Interactive scrolling timeline for company history.
- **Neural Architects**: Holographic team profile cards with clearance levels.
- **Robodino Chat**: A fully functional AI chatbot simulation with typing indicators and auto-responses.
- **Bot Defense**: A playable mini-game embedded in the footer.
- **Advanced Visuals**: High-level "Illusion Blobs" background and holographic 3D tilt cards.
- **Premium Motion**: Cinematic "Slide-Blur" page transitions.
- **Leadership Team**: Dynamic "Neural Architects" section for company leadership.

### 🎨 **Visual Identity**
- **Cyberpunk 3D Aesthetic**: Unified visual style using high-quality 3D neon renders.
- **Local Assets**: All images are locally optimized for maximum performance and zero external dependencies.
- **Consistent Branding**: Neon Blue (`#00AFFF`) and Dark Tech (`#0B0F14`) palette applied across all assets.

### ⚡ **Technical Excellence**
- **Performance**: 100/100 Lighthouse score potential with optimized `next/image` assets.
- **SEO Ready**: Comprehensive OpenGraph, Twitter Cards, and JSON-LD metadata.
- **PWA Capable**: Manifest file included for installable web app support.
- **Strict Typing**: 100% TypeScript coverage with zero `any` types.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)

## 📦 Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/botmate.git
    cd botmate
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view the project.

4.  **Build for production**:
    ```bash
    npm run build
    npm start
    ```
